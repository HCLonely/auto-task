Automatically complete giveaway tasks.

[Document](https://auto-task-doc.js.org/)

## Installation

1. Before: Please install [Tampermonkey](https://www.tampermonkey.net/) extension before installing this script
2. Installation: [Test version](https://auto-task-test.hclonely.com/auto-task-test.user.js)  [Official version](https://auto-task.hclonely.com/auto-task.user.js)

PS: New features and new website additions will be tested in the test version first.

## Usage

1. If you are using it for the first time, please set it up first: [Settings page of the test version](https://auto-task-test.hclonely.com/setting.html)  [Settings page of the official version](https://auto-task.hclonely.com/setting.html)
2. Open the corresponding giveaway website. If you see a few more buttons in the upper right corner, you can use it. The function of each button can be set in the setting page of the previous step.

Important:

1. Do not use version 2.0+ and version 3.0+ at the same time
2. The settings of the 2.0+ version and the 3.0+ version are not synchronized, and the 3.0+ version must be set for the first time
3. To implement discord related functions, additional auxiliary scripts need to be installed: [Test version](https://auto-task-test.hclonely.com/auto-task-helper.user.js)  [Official version](https://auto-task.hclonely.com/auto-task-helper.user.js)
